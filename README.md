# IoT Disaster Response System

This project is based on ESP32 microcontroller and sensors to detect disasters like fire, gas leakage, and temperature changes.

## Components Used
- ESP32
- DHT11 Sensor
- Gas Sensor
- Flame Sensor
- LCD Display
- Buzzer

## Features
- Real-time monitoring
- Fire and gas detection
- Alerts using Blynk app
- LCD display output

## Applications
- Smart homes
- Industrial safety
- Disaster management
